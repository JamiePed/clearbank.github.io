---

This webhook confirms the creation of a virtual account.

#### Request body

```json
{
  "Type":"VirtualAccountCreated",
  "Version":2,
  "Payload": {...}
  "Nonce":
}
```

#### Payload
| Element                     | Description                                                                                                                                                                                        |
|-----------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `ClientMoneyAccountId`      | A unique ID to identify the real (pooled) account.<br /> Type: `GUID`<br /> Max. length: `36`                                                                |
| `VirtualAccountIdentifiers` | **Required**<br />Virtual account identifer object. It contains `IBAN`, `BBAN`, `AccountName` and `AccountHolderLabel`. <br /><br />`IBAN`: International Bank Account Number associated with the account.<br /> Type: `string`  <br /> Max. length: `34` <br /><br />  `BBAN`: Basic Bank Account number associated with the account.<br /> Type: `string` <br /> Max. length: `30` <br /><br />`AccountName`: Name of the account. <br /> Type: `string`<br /> Max. length: `128`<br /><br />`AccountHolderLabel`:The account description. <br /> Type: `string` <br /> Max. length: `100` |
| `VirtualAccountId`          | A unique ID to identify the virtual account <br />Type: `GUID` <br /> Max. length: `36`                                                                                                           |
| `TimestampCreated`          | Date and Time the virtual account was created<br /> Time Zone: `UTC`<br /> Type: `DateTime`                                                                                                         |
| `Nonce`                 | Cryptographically secure number generated by ClearBank® for every single webhook. A nonce number is generated randomly and should not be used as a check for duplication. |
